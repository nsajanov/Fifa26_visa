# ⚽ WC 2026 Playoff Predictor — Telegram-бот (кнопки в чате)

Прогноз плей-офф ЧМ-2026 прямо в чате. **Без мини-приложения и сайта.** Игрок жмёт START → бот сразу показывает матч → тапает, кто проходит → так до чемпиона (~1 минута). Раз в день бот сам подтягивает результаты и постит лидерборд в группу.

## Как для игрока
1. Открыл бота → нажал **START** (один раз — правило Telegram).
2. Сразу появляется первый матч с двумя кнопками-командами. Тапает, кто проходит — идёт следующий матч, и так до 🏆.
3. `/restart` — переделать · `/mybracket` — мой прогноз · `/leaderboard` · `/facts`.

## Очки
1/16 **+1** · 1/8 **+1.5** · 1/4 **+2** · 1/2 **+3** · Финал/Чемпион **+5**. Максимум **47**.

## Файлы
`bot.py` — бот · `bracket.py` — 16 пар Round of 32 и сетка · `sheets.py` — хранение · `results_api.py` + `fixtures.py` — авто-результаты (football-data.org).

> Мини-приложения (webapp) больше нет. Если в репозитории осталась папка `webapp/` — удали её, а на Railway убери переменную `WEBAPP_URL`.

## Запуск (Railway)
1. **@BotFather** → `/newbot` → токен.
2. Залей файлы из этой папки в GitHub-репозиторий.
3. railway.app → New Project → Deploy from GitHub → выбери репозиторий.
4. **Variables**:
   - `BOT_TOKEN` — токен
   - `ADMIN_IDS` — твой id (спроси бота `/id`)
   - `GROUP_CHAT_ID` — id группы для авто-постинга (узнать: `/chatid` в группе)
   - `FOOTBALL_DATA_TOKEN` — бесплатный ключ football-data.org (для авто-результатов; без него используй `/aw`)
   - `SYNC_HOUR=10`, `POST_HOUR=11`, `TZ_OFFSET=5` (Алматы)
   - `SPREADSHEET_ID`, `GOOGLE_CREDENTIALS_JSON` — опционально (иначе данные в `local_db.json`)
5. В логах: `Scheduled: sync 10:00 / post 11:00 Almaty.` и `Bot running…`.

## Реальные пары плей-офф
Заданы в `bracket.py` → `R32_PAIRS` (в порядке сетки). Если что-то изменится — поправь там же.

## Админ
- `/setdeadline 2026-06-28 16:00` — закрыть приём (по Алматы).
- `/sync` — подтянуть результаты сейчас · `/aw 1 Brazil` — вручную победитель матча (1–31) · `/post` — таблица в группу.

## Авто (само)
- 10:00 Алматы — подтяг результатов; 11:00 — пост лидерборда + таблиц групп в группу.
